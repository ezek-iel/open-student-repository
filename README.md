# Open Student Repository (OSR)

The **Open Student Repository** is an open-source directory of free and discounted student materials. Our goal is simple: help students save money while sourcing academic resources.  

---

> [!TIP]
> OSR is built **by students, for students**. We encourage student developers to contribute, improve, and expand this project.  
> For details on contributing, see the [Contribution Guide](CONTRIBUTING.md).
